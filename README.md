# Java Mastery (Capgemini Training)

Welcome to the **Java Mastery** repository! This codebase serves as a comprehensive collection of my learning journey, featuring topic-wise classroom exercises, hands-on assignments, and mini-projects developed during my Java training program.

## 📚 Repository Structure

The repository is organized into two primary directories to separate guided classroom code from independent assignments:

### 1. Classroom Exercises (`/Classroom`)
Contains topic-oriented implementations demonstrating core Java concepts, tools, and frameworks.

- **`01-core-java`**: Fundamentals of Java programming.
- **`02-oops`**: Object-Oriented Programming principles.
- **`03-collections`**: Practical usage of the Java Collections Framework.
- **`04-funtional-programming`**: Lambdas, Streams, and modern Java features.
- **`05-exception-handling`**: Robust error and exception management.
- **`06-file-handling`**: Reading, writing, and manipulating files.
- **`07-multithreading`**: Concurrency and thread management.
- **`08-jdbc`**: Database connectivity using JDBC.
- **`09-junit5`**: Unit testing best practices with JUnit 5.
- **`10-hibernate`**: Object-Relational Mapping (ORM) using Hibernate.
- **`11-jpa`**: Implementations using the Java Persistence API.
- **`12-Springcore`**: Dependency Injection and Inversion of Control with Spring Core.

### 2. Practical Assignments (`/Assignment`)
Real-world problems, mini-projects, and exercises implementing the concepts covered during classroom sessions.

- **Core Java Assignment & Exceptions**: Foundational problem-solving and error-handling exercises.
- **Employee_Management_System**: A console-based application for managing employee data.
- **JDBCApplication**: An application establishing database connections to perform CRUD operations.
- **HibernateAssignment, Hibernet-JPA & JPAAsignment**: Advanced database interaction tasks using ORM techniques.
- **Mapping**: Exercises focused on entity relationships (e.g., One-to-One).
- **ProductManagement & stockmanagement**: Mini-projects implementing business logic and inventory tracking.

## 🛠️ Technology Stack

- **Language:** Java
- **Testing:** JUnit 5
- **Database / ORM:** JDBC, Hibernate, JPA
- **Frameworks:** Spring Core
- **Build Tool:** Maven

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/i-m-javed/capgemini-training.git
   ```
2. **Open in IDE:** Open the repository folder in your preferred Java IDE (e.g., IntelliJ IDEA, Eclipse, or VS Code).
3. **Explore Modules:** Navigate to the specific module (e.g., `Classroom/08-jdbc`) and run the respective `Main` class or test cases.
4. **Dependencies:** Ensure your IDE is set up to import Maven dependencies correctly.
